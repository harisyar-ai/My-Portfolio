Developer Journey

From writing first lines of Python to building full-stack government platforms — here's how it unfolded.

Phase 01 — Foundation
HTML & CSS — Building the Visual Web
Started with the fundamentals: semantic HTML structure, CSS layouts, flexbox, grid, animations, and responsive design. Learned to craft clean, accessible, and visually polished interfaces from scratch — the skill that later made every project feel professional rather than generic.
Phase 02 — Programming Core
Python — The Language of Intelligence
Self-taught Python from the ground up. Mastered data structures, OOP, file handling, and the scientific ecosystem — NumPy, Pandas, Matplotlib. Python became the backbone of every project that followed, from ML pipelines to full web backends.
Phase 03 — Backend & Web
Flask — Full-Stack Web Development
Moved into backend development using Python Flask. Built complete web applications with routing, authentication, SQLite databases, Jinja2 templating, file uploads, and REST APIs. Combined with HTML/CSS frontend knowledge to deliver truly full-stack solutions — both the server and the interface, solo.
Phase 04 — Machine Learning
ML Engineering — Models That Predict & Understand
Dove deep into supervised learning, regression, classification, and ensemble methods (LightGBM, Random Forest, Gradient Boosting). Built end-to-end ML pipelines: data scraping, cleaning, feature engineering, model comparison, SHAP explainability, and Streamlit deployment. Achieved R² scores up to 97.48% on real-world Pakistani market data.
Phase 05 — Deep Learning & AI
TensorFlow / Keras, NLP & Computer Vision
Expanded into deep learning with TensorFlow and Keras — CNNs, NLP pipelines (TF-IDF, cosine similarity, Hugging Face transformers), and digital image processing. Currently a Research Assistant at a dedicated DIP Lab, applying these skills to real research.
Phase 06 — Real-World Impact
IDBPM — Government MIS Platform (Current)
The culmination of the full-stack journey: designed and developed a production-grade governmental Management Information System for districts across the region — combining Python Flask backend, custom HTML/CSS frontend, SQLite database, Excel data processing, AI-driven prioritization, and multi-role authentication. Built from scratch. Deployed for real use.

 Featured Project

IDBPM — District Planning & Management Information System

A full-stack governmental MIS platform designed for all districts in the region. Built from scratch to replace a dated, limited legacy system — with intelligent prioritization, multi-role access, and a modern interface that actual government officers use every day.

Multi-role authentication system (Admin, Officers, Viewers)
AI-driven scheme prioritization — scores & ranks development projects by criteria
Excel/CSV upload → parse → display → edit → re-prioritize pipeline
Interactive data grids with column filtering (Excel-style AutoFilter)
Upload Data section for needlists & general district data management
MTDP documents, sectoral profiles, VCNC management across tehsils
Government / MIS

The previous system was basic, fragmented, and couldn't handle the complexity of real district-level planning. I redesigned it completely — adding a proper Flask backend with SQLite, a clean HTML/CSS interface matching official branding, and a novel AI prioritization engine that scores schemes across multiple criteria (beneficiary households, estimated cost, sector, VCNC) to help planners make better decisions faster.

Built solo over months of continuous iteration, this is the project where all skills converged — full-stack web, data processing, UX design, and real-world deployment.

Python Flask HTML / CSS SQLite openpyxl Pandas Jinja2

Content-based book recommendation engine using TF-IDF and cosine similarity across 10,000+ books. Analyzes title, author, description, and genres to surface meaningful matches — no ratings, no history needed. Includes a custom cover image scraper and mobile-responsive Streamlit UI.
